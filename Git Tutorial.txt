Check status: 
    git status
Git Initialization:
    git init 
    git add .
    git commit -m "comment"
    git remote add origin REPO-URL
    git push -u origin master / git push --force origin master

If folder already has repository before and cant access:
    cd thatFolder
    rm -rf .git

    PULL: git pull origin master

    PUSH: git add .
          git commit -m "comment"
